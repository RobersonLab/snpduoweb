Requirements

gcc

Perl
Perl CGI Module

Recommendations

Perl Image::Magick Module -- Greatly increases performance of JPEG creation speed

Upload directory on same disk partition as HTML output -- Allows Perl rename to move files versus command-line fork to mv

SNPduo++

A command-line based C++ program for performing SNPduoWeb IBS, Mean_IBS, and Standard Deviation of IBS on large datasets in ped/map or tped/tmap (PLINK compatible) format. Works under Windows and Linux. **Windows requires manual compilation.
